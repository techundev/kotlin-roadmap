# 🟣 Kotlin Roadmap — De cero a avanzado

Repositorio de aprendizaje progresivo de Kotlin. Cada sección contiene teoría, ejemplos ejecutables y un reto práctico con su resolución.

---

## ¿Cómo usar este repositorio?

1. Sigue las secciones **en orden** — cada una construye sobre la anterior.
2. Lee el `README.md` de cada tema.
3. Intenta el **reto** por tu cuenta antes de ver la solución.
4. Revisa el `SOLUCION.md` solo después de intentarlo.
